SeeFood

Food Image Recognizer

Code

A food image classifier that uses convolutional neural networks (Keras framework). Inspired from the app "SeeFood", as shown on HBO's Silicon Valley. Also has a web interface that can download images from any URL and classify them.

Sentiment

Twitter Sentiment Analyzer

Code

A twitter sentiment analyzer that used support vector machines and natural language processing (scikit-learn framework and NLTK). Used the word frequency map method to build vectors (after stopword removal, POS tagging and spelling correction) and the one versus all SVM method.

Document Classifier

Finance Document Categorizer

Code

A document categorizer that uses support vector machines and neural networks (used Neuroph, and libsvm framework). Used the word frequency map method to build vectors (after stopword removal, and special character removal) and the one versus all SVM method. On the other hand, a multi-perceptron neural network was used. Achieved a 100% accuracy on BBC and 89% on the Heckyl Database.
